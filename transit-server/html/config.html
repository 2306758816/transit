配置管理

Transit 转发配置管理页面

请谨慎操作!

请勿修改他人记录信息 ! 请勿添加重复信息 !

Example :
    1 To 1 (一个前端机器对应一个后端机器) :
        127.0.0.1 -> 127.0.0.1:8001

    N To 1 (多个前端机器对应一个后端机器, 以分号分隔) :
        127.0.0.1;127.0.0.2 -> 127.0.0.1:8001

    tips : 不支持 N To N (多个前端机器对应多个后端机器)
